🚀 FASTEST WAY TO DEPLOY:

1. Create GitHub repository: https://github.com/new
   Name: gym-pro-whatsapp

2. Run these commands (replace YOUR_USERNAME):
   
   git remote add origin https://github.com/YOUR_USERNAME/gym-pro-whatsapp.git
   git branch -M main
   git push -u origin main

3. Deploy on Render: https://render.com
   - Sign up with GitHub
   - New + → Blueprint
   - Select gym-pro-whatsapp
   - Click Apply

4. Initialize database (in Render web service shell):
   
   npx prisma db push

5. Connect WhatsApp bot:
   - Go to worker service logs
   - Scan QR code with WhatsApp

DONE! 🎉
